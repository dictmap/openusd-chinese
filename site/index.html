USD 是一个用于创作、组合与读取层级化组织的场景描述的系统。

USD is a system for authoring, composing, and reading hierarchically organized scene description.

USD 由一组模块组成，可以在数字内容创作应用之间，以可扩展方式编码并交换静态以及按时间采样的 3D 几何和着色数据。面向特定领域的 schema 模块，会在 USD 与具体领域无关的核心之上定义几何和着色的编码方式。

USD comprises a set of modules that scalably encode and interchange static and time-sampled 3D geometry and shading data between Digital Content Creation applications. Domain-specific schema modules define the geometry and shading encoding atop USD's domain-agnostic core.
