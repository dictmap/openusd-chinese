Overview UsdUtils contains utility classes and functions for managing, inspecting, editing and creating USD assets. The high-level grouping of classes and functions is as follows: Authoring provides functionality for copying metadata between layers. Coalescing diagnostic delegate provides a utility to aggregate warnings and statuses emitted from Tf's diagnostic management system.
