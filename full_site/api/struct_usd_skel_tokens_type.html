原文摘录 / English Source Excerpts

UsdSkelTokens provides static, efficient TfTokens for use in all public USD API. These tokens are auto-generated from the module's schema, representing property names, for when you need to fetch an attribute or relationship directly by name, e.g. UsdPrim::GetAttribute(), in the most efficient manner, and allow the compiler to verify that you spelled the name correctly.
