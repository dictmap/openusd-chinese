原文摘录 / English Source Excerpts

Overview This library houses the definitions and implementations for all the current (and future) "Usd*" shader nodes, eg. UsdPreviewSurface, UsdUVTexture etc. The definitions are represented by a UsdShade-based shader definition file. See UsdShade Based Shader Definition. It currently only contains "glslfx" implementations for the hydra GL backend. In the (not-too-distant) future, it will contain "oso" files that provide OSL implementations of the Usd* shaders.
