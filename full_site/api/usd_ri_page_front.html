Overview For USD schemas corresponding to RenderMan concepts and plugins, see the usdRiPxr library. This module provides certain utilities for authoring USD that encodes RenderMan-specific information, and for converting between USD and Ri values and datatypes. There is no inclusion of RenderMan headers, so this module is compilable and useful regardless of whether you have or use RenderMan.
