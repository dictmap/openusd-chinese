While prims have a unique name and hierarchical identifier, it is sometimes useful to reason about the scene graph using a set of labels. def Xform "OfficeBookshelf" (apiSchemas = ["SemanticsLabelsAPI:category"])

Semantic labels have hierarchical inheritance semantics similar to primvars. However, instead of descendants strictly overriding parents, labels may be accumulated.
