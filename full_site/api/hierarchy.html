原文摘录 / English Source Excerpts

Hierarchy entries include: ExecSystem::_ChangeProcessor: Public API to deliver scene changes from ExecSystem derived classes; SdfSchemaBase::_SpecDefiner: Class that defines fields for a spec type; SdfSchemaBase::_ValueTypeRegistrar: A helper for registering value types; HdSceneIndexObserver::AddedPrimEntry: A notice indicating a prim of a given type was added to the scene; allocator : STL class; Vdf_DefaultInitAllocator : Vdf_DefaultInitAllocator; App: Creates a command line program, with very few defaults; Option_group: Extension of App to better manage groups of options; AppFriend: This class is simply to allow tests access to App 's protected functions; ArAsset: Interface for accessing the contents of an asset.
