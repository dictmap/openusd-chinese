Contents Overview Instrumentation Recording and Reporting Performance Overhead Details Overview Defines utility classes for counting, timing, measuring, and recording events. The TraceCollector class records TraceEvent objects. The TraceReporter class contains functions to generate reports on events gathered by the TraceCollector object. Instrumentation Instrumentation is done by adding TRACE macros to code. For example: #include " pxr/base/trace/trace.h "
