原文摘录 / English Source Excerpts

Overview This module provides schemas for encoding information on USD prims for client graphical user interface tools to use in deciding how to organize and present the prims in UI layouts. This information does not guide the 3D rendering of data contained in a scene, rather it's about how to visualize the nodes in the graph that comprises a UsdStage.
