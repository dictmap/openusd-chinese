This section adds Chinese-first function-index usage notes, module-reading guidance, and terminology for Class Members - Functions - U while retaining the English page name, API symbols, template parameters, code, links, and source excerpts for comparison with the official Doxygen page.

  • functions_func_u.html 是 U 段函数成员索引，条目常围绕 update/unload/unregister/use 等语义展开，同时跨越 Glf、UsdShade、Ar、HdSt、Vt、Gf、UsdStage、Vdf 和 Trace。
