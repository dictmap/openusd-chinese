原文摘录 / English Source Excerpts

TsTest is a framework that validates, graphs, and compares spline evaluations. It is used to test Ts itself, but it also has other "backends", which perform evaluation using other engines. TsTest lives inside the Ts library. All files prefixed with tsTest, wrapTsTest, or TsTest are part of the TsTest framework. They fall into these categories: Framework.
