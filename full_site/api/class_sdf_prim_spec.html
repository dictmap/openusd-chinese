原文摘录 / English Source Excerpts

Represents a prim description in an SdfLayer object. Every SdfPrimSpec object is defined in a layer. It is identified by its path ( SdfPath class) in the namespace hierarchy of its layer. SdfPrimSpecs can be created using the New() method as children of either the containing SdfLayer itself (for "root level" prims), or as children of other SdfPrimSpec objects to extend a hierarchy.
