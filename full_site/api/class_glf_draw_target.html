原文摘录 / English Source Excerpts

A class representing a GL render target with mutliple image attachments. A DrawTarget is essentially a custom render pass into which several arbitrary variables can be output into. These can later be used as texture samplers by GLSL shaders. The DrawTarget maintains a map of named attachments that correspond to GL_TEXTURE_2D mages.
