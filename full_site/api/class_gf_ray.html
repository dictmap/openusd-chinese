原文摘录 / English Source Excerpts

Basic type: Ray used for intersection testing. This class represents a three-dimensional ray in space, typically used for intersection testing. It consists of an origin and a direction. Note that by default a GfRay does not normalize its direction vector to unit length. Note for ray intersections, the start point is included in the computations, i.e., a distance of zero is defined to be intersecting. Definition at line 44 of file ray.h .
