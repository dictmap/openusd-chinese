原文摘录 / English Source Excerpts

The Kind library provides a runtime-extensible taxonomy known as "kinds". Kinds are just TfToken symbols, but the KindRegistry allows for organizing kinds into taxonomies of related/refined concepts, and the KindRegistry::GetBaseKind() and KindRegistry::IsA() queries enable reasoning about the hierarchy and classifying objects by kind.
