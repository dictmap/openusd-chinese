原文摘录 / English Source Excerpts

Token for efficient comparison, assignment, and hashing of known strings. A TfToken is a handle for a registered string, and can be compared, assigned, and hashed in constant time. It is useful when a bounded number of strings are used as fixed symbols (but never modified). For example, the set of avar names in a shot is large but bounded, and once an avar name is discovered, it is never manipulated.
