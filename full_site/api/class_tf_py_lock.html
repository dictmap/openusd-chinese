原文摘录 / English Source Excerpts

Convenience class for accessing the Python Global Interpreter Lock. The Python API is not thread-safe. Accessing the Python API from outside the context of Python execution requires extra care when multiple threads may be present. There are various schemes for how this should be done, and the conventions have been changing with Python versions through 2.X.
