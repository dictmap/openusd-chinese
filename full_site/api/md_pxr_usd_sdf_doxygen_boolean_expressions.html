Boolean Expression Language The syntax for boolean expression is intended to be concise and recognizable to those familiar with C/C++. An expression can be a constant value, a variable, or operations applied to one or more sub-expressions. The final result of evaluating an expression is always a boolean value. Constants The simplest allowed expressions are constant values.
