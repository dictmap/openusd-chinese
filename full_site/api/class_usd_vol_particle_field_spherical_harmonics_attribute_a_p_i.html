A ParticleField related applied schema that provides spherical harmonics attributes to define the radiance of the particles. The spherical harmonics degree is constant across all the particles in the ParticleField. Attributes are provided in both float and half types for some easy data footprint affordance, data consumers should prefer float version if available.
