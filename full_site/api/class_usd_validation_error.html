UsdValidationError is an entity returned by a validation task, which is associated with a UsdValidationValidator . A UsdValidationError instance contains important information, like: Name - A name the validator writer provided for the error. This is then used to construct an identifier for the error. UsdValidationErrorType - severity of an error, UsdValidationErrorSites - on what sites validationError was reported by a validation task, Message - Message providing more information associated with the error.
