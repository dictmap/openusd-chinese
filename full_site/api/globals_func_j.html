  • `JsFindValue()` 来自 `utils.h`，用于在 JSON-like 数据结构中查找值；它与 parse/write 函数属于同一 Js 工具域，但职责不同。
  • `JsParseStream()` 与 `JsParseString()` 来自 `json.h`，分别处理流和字符串输入；`JsWriteToStream()`、`JsWriteToString()`、`JsWriteValue()` 则处理输出和序列化。
