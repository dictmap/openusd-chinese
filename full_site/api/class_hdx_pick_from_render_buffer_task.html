原文摘录 / English Source Excerpts

A task for running picking queries against pre-existing id buffers. This task remaps the "pick frustum", provided by HdxPickTaskContextParams , to the camera frustum used to generate the ID buffers. It then runs the pick query against the subset of the ID buffers contained by the pick frustum. Definition at line 61 of file pickFromRenderBufferTask.h .
