This tutorial aims to show USD in an extremely simplified pipeline. The files for this tutorial live in USD/extras/usd/tutorials/endToEnd .

assets: This folder contains some files that are used in the tutorial. It also contains some of the geometry in case you do not want to author them yourself.

scripts: Some generic scripts we use in the tutorial. We’ve tried to document them. Feel free to poke around, or run them with -h .
