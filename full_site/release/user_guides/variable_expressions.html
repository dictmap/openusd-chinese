  • `expressionVariables` 定义在 layer metadata 中，常见值类型包括 string、bool、int64、数组和 `None`。这些变量会在表达式求值时参与替换。
  • USDA 示例中的反引号表达式必须保留原样，例如 `@`"${ASSET_PATH}/extraAssets.usda"`@` 和 ```${VARIANT_CHOICE}``` 这类语法决定解析行为。

原文摘录 / English Source Excerpts

Variable expressions are specially formatted strings that contain expressions that USD will evaluate at runtime. The expressions can contain variable data and logic used to dynamically create or change values in your USD scene. Use variable expressions to easily configure your scene without having to make extensive scene data modifications.

Variable expressions use named expression variables that can be easily defined and changed in layer metadata. Expression variables are defined in a “expressionVariables” dictionary metadata field in a layer.
