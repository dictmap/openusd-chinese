  • `GeometryLight` 表示从 geometric prim 向外发光的灯光，通常关联到 `Mesh`；官方摘录明确说明该 schema 已 deprecated，应改用 applied 到 Mesh 的 `MeshLight`。

Light emitted outward from a geometric prim, typically a Mesh. This schema is deprecated, use MeshLight applied to a Mesh instead.

Relationship to the geometry to use as the light source.

The shader ID for a GeometryLight. USD will also register a Sdr shader node with a “GeometryLight” identifier and the source type “USD” to correspond to the light’s inputs
