原文摘录 / English Source Excerpts

The UsdLux schema domain contains schemas for working with lights and light-related behaviors (e.g. shadows, light filters, etc.). UsdLux is designed to provide representations of lighting used in a typical 3D graphics setting in an efficient and portable way.

UsdLux includes several schemas that provide the following features:

Represent boundable lights (e.g. SphereLight)
