原文摘录 / English Source Excerpts

Defines a base applied schema for ParticleField radiance definition data.

This is a base schema that ParticleField characteristics schemas typically will have as a built-in applied schema. This allows validation to verify that an applied schema that provides radiance definitions is always present for a ParticleField.
