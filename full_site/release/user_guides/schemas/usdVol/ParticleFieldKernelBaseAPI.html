原文摘录 / English Source Excerpts

Defines a base applied schema for ParticleField kernel data. The ParticleField kernel defines the spatial basis function for every particle.

This is a base schema that ParticleField characteristics schemas typically will have as a built-in applied schema. This allows validation to verify that an applied schema that defines kernel data is always present for a ParticleField.
