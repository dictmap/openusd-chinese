  • `usdMedia` 应先按领域入口阅读：它补充 asset browser、DCC 工具和播放端需要的媒体语义，不直接修改 mesh、material 或 lighting schema。

原文摘录 / English Source Excerpts

The UsdMedia domain provides ways to associate various forms of media, such as audio ( SpatialAudio ) or thumbnails ( AssetPreviewsAPI ) with assets.

SpatialAudio allows for ambient audio playback or audio playback from a specific location. It also allows setting various playback options.

AssetsPreviewsAPI allows for setting one or more thumbnails for an asset. The thumbnails could be pre-rendered images of the asset that could be presented in the asset browser of a DCC tool, or in a system asset browser.
