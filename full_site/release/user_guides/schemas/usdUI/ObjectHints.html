原文摘录 / English Source Excerpts

ObjectHints provide UI hints for all USD objects that can be presented in a UI. ObjectHints cover general UI features present in prims and properties, such as the user-facing name of the object shown in a UI, or whether the object is hidden in a UI.

The following simple example shows a prim and an attribute with ObjectHints for displayName and hidden . In the UI for a DCC tool, the “Placeholder” prim would show “ModelA Placeholder” for the name and “Model is annotated” for the name for the isAnnotated attribute. The isAnnotated would be hidden and not displayed in the UI by default.
