原文摘录 / English Source Excerpts

The Alembic USD plugin is not built by default. To enable it, the PXR_BUILD_ALEMBIC_PLUGIN option must be specified when running CMake. Check the 3rd Party Library and Application Versions document for the Alembic library versions that have been used for testing.

For more information see our page on Advanced Build Configuration .

As shipped, USD supports its own text (usda) and binary (usdc) file formats. However, it is possible for USD to support additional formats by providing it with a custom file format plugin. One of these plugins, usdAbc, which provides the ability to interact with Alembic files, is included in the pxr/usd/plugin directory of the USD distribution. In this example, we’ll demonstrate the extended USD functionality using the existing USD tools.
